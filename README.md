# AWSネットワーク入門
## コマンド一覧
### 定義内容のチェック
```
$ terraform plan
```

### 定義の適応
```
$ terraform apply
```

### フォーマット
```
$ terraform fmt 　// currentディレクトリのみ
$ terraform fmt -recursive // サブディレクトリ内も整形

```
